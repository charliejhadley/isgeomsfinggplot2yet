Is geom_sf in ggplot2 yet?


NO.

The version of ggplot2 on CRAN does not yet have support for visualising sf objects.

Until it does, you could use the dev version - see https://github.com/tidyverse/ggplot2

Why should I care?

sf is very young (initial development started in 2016) but is highly recommended as the best solution for working with GIS data in R:

  • It fits into the tidyverse (dplyr etc) workflow
  • It simplifies importing GIS datasets
  • It has built in tools for converting to/from other formats, including sp
  • It has built in tools for geomtric calculations (great circles etc)
  • It has the backing of the R Consortium!

Read about sf here https://r-spatial.github.io/sf/
